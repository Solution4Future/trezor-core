##Build instructions

###Linux

####Debian/Ubuntu

```
sudo dpkg --add-architecture i386
sudo apt-get update
sudo apt-get install libsdl2-dev:i386
make build_unix
```

####Fedora

```
sudo yum install SDL2-devel.i686
make build_unix
```

####openSUSE

```
sudo zypper install libSDL2-devel-32bit
make build_unix
```

###OS X

```
brew install --universal sdl2
make build_unix
```
